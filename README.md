**Diabetes Prediction using Machine Learning**
**Overview -** This repository presents an end-to-end machine learning project focused on predicting diabetes using patient medical data. The project demonstrates a complete ML workflow—from data preprocessing and exploratory analysis to model training, evaluation, and comparison—following industry best practices.

The aim is to showcase practical machine learning skills, clean code organization, reveal model behavior, and decision-making based on evaluation metrics.

**Key Highlights**
      End-to-end ML pipeline implementation
      Comparison of baseline and ensemble models
      Industry-standard evaluation metrics
      Modular and scalable project structure

**Models Implemented**
      Logistic Regression – Baseline linear classification model
      Random Forest Classifier – Ensemble-based non-linear model

**Dataset**
      Dataset: PIMA Indians Diabetes Dataset
      Description: Medical diagnostic measurements for diabetes prediction

**Key Features:**
      Glucose Level
      Blood Pressure
      Body Mass Index (BMI)
      Insulin
      Age
      Skin Thickness

**Tech Stack**
      Language: Python
      Libraries:
      Pandas, NumPy
      Scikit-learn
      Matplotlib, Seaborn

**Project Workflow**

      Data Cleaning & Handling Missing Values
      Exploratory Data Analysis (EDA)
      Feature Scaling & Transformation
      Model Training
      Model Evaluation & Comparison
      Performance Analysis

**Model Performance**
      Model	Accuracy
      Logistic Regression	75%
      Random Forest	76%

**Evaluation Metrics Used:**
      Accuracy
      Precision
      Recall
      F1-Score
      Confusion Matrix

**Conclusion -** The Random Forest model demonstrated slightly better performance compared to Logistic Regression, indicating its ability to capture non-linear patterns in medical data. This project highlights the importance of model selection and evaluation when working with real-world datasets.
